# Hello-World（免费VPN）


1.蓝灯：

优点：免费。
缺点：速度慢，不稳定，有流量限制：每月800MB，不支持iPhone、iPad。
可用于：Windows电脑、Mac电脑、安卓手机及平板。
中文支持：支持中文。
下载地址：蓝灯官网（需翻墙），Github（不需要翻墙）

蓝灯（Lantern）是一个代理类翻墙工具，分为专业版和免费版。专业版没有流量和速度限制，并且支持多台设备同时连接。专业版只能按年购买，不能按月购买。蓝灯免费版可以免费使用，但每月的流量限制是800MB，流量用完后连接速度会降到很低（20KB/s），看文字内容基本还可以，但图片打开非常慢，就更别提视频了。但是对于偶尔翻墙、对速度和稳定性要求不好的网友来说，蓝灯免费版是一个不错的选择。

蓝灯官方网站（需要翻墙）：https://getlantern.org/
蓝灯VPN官方论坛（不需要翻墙）网址：

蓝灯的Windows下载（不用翻墙）https://raw.githubusercontent.com/getlantern/lantern-binaries/master/lantern-installer.exe
蓝灯安卓下载（不用翻墙）https://raw.githubusercontent.com/getlantern/lantern-binaries/master/lantern -installer.apk




2.赛风：

优点：简单，好操作，无流量限制。
缺点：速度慢，不稳定，不支持苹果设备，在某些地区遭屏蔽。
可用于：Windows电脑、安卓手机及平板。
中文支持：支持中文。

赛风（Psiphon）是一款老牌翻墙软件，其最新版本利用VPN、SSH和HTTP代理软件提供翻墙服务。用户可以选择位于不同国家的服务器。但目前赛风只支持 Windows 电脑和安卓手机及平板，有时会出现稳定性、速度不理想的情况，而且在某些地区已遭到屏蔽。

官网（需要翻墙）：https://psiphon.ca/zh/download.html
镜像（不需要翻墙）：https://s3.amazonaws.com/0ozb-6kaj-r0p8/zh/download.html


3.Tor

优点：无流量限制。
缺点：速度很慢，不稳定，初始设置比较麻烦，翻墙时需使用专门的浏览器Tor Browser，不支持iPhone、iPad。
可用于：Windows电脑、Mac电脑、Linux电脑、安卓手机及平板。
中文支持：网站无中文版，但软件有中文版。

官网（需要翻墙）：https://www.torproject.org/download/

Tor是一个老牌翻墙软件，曾一度被封，但最新版本在中国目前可用。Tor的安装、设置并不特别简单直观，需要参考相应教程。使用Tor翻墙时用户需要使用专门的浏览器Tor Browser，可能很多人对此感觉不是非常习惯。另外用Tor翻墙速度经常很慢，而且Tor再度被封的风险还是很高的。
